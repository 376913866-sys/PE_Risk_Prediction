## Ethics & Data Privacy Statement

This project is for academic research and educational demonstration only.

- No real patient-level data are included in this repository.
- The trained model was developed using de-identified retrospective data.
- The web application does not store, transmit, or log any user inputs.
- This tool is NOT intended for clinical diagnosis or medical 
decision-making.

Users are responsible for obtaining appropriate ethics approval before 
applying similar models to real-world clinical data.
## 伦理与数据隐私说明（中文）

本项目仅用于科研与教学演示目的。

- 本仓库不包含任何真实患者的原始数据；
- 模型基于去标识化的回顾性数据训练；
- 在线预测页面不会保存、上传或记录用户输入；
- 本工具不用于临床诊断或实际医疗决策。

如将类似模型用于真实临床数据，需自行获得伦理委员会批准。
